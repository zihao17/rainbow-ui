# Button 按钮

按钮用于开始一个即时操作。

## 组件引入方式

```jsx
import { Button, ButtonType, ButtonSize } from 'rainbow-ui';
```

## 组件使用示例

```jsx
<Button btnType={ButtonType.Primary} size={ButtonSize.Large}>
  按钮
</Button>
```

## 按钮类型

Button 组件支持多种类型的按钮，包括默认按钮、主要按钮、警告按钮、危险按钮和链接按钮。

<Canvas of={ButtonStories.Basic} />

## 按钮尺寸

Button 组件支持三种尺寸：大、中、小。

通过设置 `disabled` 属性可以禁用按钮。

## 代码示例

### 主要按钮

```jsx
import { Button, ButtonType } from 'rainbow-ui';

// 主要按钮
<Button btnType={ButtonType.Primary}>主要按钮</Button>;
```

### 警告按钮

```jsx
import { Button, ButtonType } from 'rainbow-ui';

// 警告按钮
<Button btnType={ButtonType.Warning}>警告按钮</Button>;
```

### 大号按钮

```jsx
import { Button, ButtonType, ButtonSize } from 'rainbow-ui';

// 大号主要按钮
<Button btnType={ButtonType.Primary} size={ButtonSize.Large}>
  大号主要按钮
</Button>;
```

### 禁用按钮

```jsx
import { Button, ButtonType } from 'rainbow-ui';

// 禁用主要按钮
<Button btnType={ButtonType.Primary} disabled>
  禁用主要按钮
</Button>;
```

### 链接按钮

```jsx
import { Button, ButtonType } from 'rainbow-ui';

// 链接按钮
<Button btnType={ButtonType.Link} href="https://example.com">
  链接按钮
</Button>;
```
