AutoComplete ：

这是一个名为 `AutoComplete` 的 React 组件，用于实现输入框的自动完成功能。它接收 `fetchSuggestions`、`onSelect`、`onChange`、`renderOption` 等属性，支持 Input 组件的除 `onSelect` 和 `onChange` 之外的所有属性。`fetchSuggestions` 方法可同步或异步返回输入建议，`onSelect` 在选中建议项时触发，`onChange` 在输入框内容改变时触发，`renderOption` 可自定义渲染下拉项。组件内部维护输入值、建议列表、加载状态等，通过防抖处理输入值，支持键盘上下键选择建议项和回车键确认选择，点击外部区域可关闭下拉列表，下拉列表有加载动画，还能通过过渡动画展示和隐藏。 


这是一个名为 AutoComplete 的 React 组件，用于实现输入框自动完成功能。它能根据用户输入，通过同步或异步的 fetchSuggestions 方法获取建议数据，展示在下拉列表中。支持键盘事件选择，如上下键切换、回车键确认、ESC 键关闭下拉列表。用户可通过 onSelect 回调处理选择项，用 onChange 监听输入变化，还能利用 renderOption 自定义下拉项渲染，同时具备防抖、点击外部关闭下拉列表及加载状态显示等功能，其依赖了 classNames 库及组件库内的 Input、Icon、Transition 组件和自定义的 useDebounce、useClickOutside 钩子 。




select： 


这是一个 Select 下拉选择器组件，用于替代原生选择器或实现优雅的多选功能。它支持单选和多选两种模式，单选时选项可设置为禁用状态 ；多选时可选择多个条目。input 框不能手动输入，点击选项自动填入对应 value 。多选模式下，点击选项添加或取消选择，点击已选 tag 上的 X 可删除该项。具备 onVisibleChange（下拉菜单显示隐藏时调用 ）、onChange（选中值变化时触发，携带当前选中项参数 ）等回调函数，需合理设计语义化的组件结构。





Upload：

这是一个名为 Upload 的 React 组件，用于实现文件上传功能。可通过点击或拖拽方式上传文件，必选参数 action 指定上传地址。支持设置默认文件列表、上传前钩子、上传中进度回调、成功与失败回调、状态改变回调及文件移除回调等。能配置请求头部、文件字段名、额外参数、是否携带 cookie、接受的文件类型、是否支持多选及是否支持拖拽上传等，内部通过 axios 处理上传逻辑，并结合 UploadList 和 Dragger 组件展示文件列表与实现拖拽上传功能 


Form：



这是一套用于表单处理的 React 组件库，包含 Form 和 FormItem 组件。Form 组件用于创建表单，可设置表单名称、初始值，有提交成功和失败的回调函数。它通过 FormContext 传递表单状态和操作方法，支持函数式子组件渲染。FormItem 组件是表单的子项，用于包裹表单字段，可设置字段名、标签、值属性名、触发事件、校验规则等。它从 FormContext 获取状态，能自动处理字段值更新和校验，可显示字段错误信息，默认配置了值属性名、触发事件和校验触发时机。
