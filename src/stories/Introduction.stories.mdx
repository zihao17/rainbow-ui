## 🌈 设计原则

Rainbow UI 基于以下设计原则创建：

- **美观** - 具有现代感的视觉设计，色彩鲜明
- **一致性** - 组件间保持一致的设计语言和交互模式
- **灵活性** - 提供丰富的属性和配置选项
- **可访问性** - 符合 WCAG 2.1 标准，对所有用户友好
- **性能优先** - 轻量且高效的实现

## 🚀 开始使用

安装 Rainbow UI：

```bash
npm install rainbow-ui
# 或者
yarn add rainbow-ui
```

导入和使用组件：

```jsx
import { Button, Alert } from 'rainbow-ui';

function App() {
  return (
    <div>
      <Alert type="info">欢迎使用 Rainbow UI!</Alert>
      <Button type="primary">开始使用</Button>
    </div>
  );
}
```
